Utilisez les flèches pour vous deplacer et la touche entrée pour selectionner dans le menu.

La fenêtre est trop petite pour afficher le jeu

Lorsqu'une partie est lancée, un cadrillage de 600 zones de couleurs differentes est généré aléatoirement, les zones marquées d'un carré blanc vous appartiennent. Vous pouvez changer la couleur de vos zones avec les boutons de couleurs. Les zones adjacentes et de même couleur que celles qui vous appartiennent deviennent votre. Le but est de vous aproprier la totalité du quadrilage en un nombre de changement de couleur impartie selon le niveau.
